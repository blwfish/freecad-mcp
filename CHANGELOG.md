## [5.0.0] - 2026-04-30

Initial public release. 32 MCP tools covering:

### Features
- Parametric sketch and PartDesign workflow (sketch → pad → pocket → fillet/chamfer)
- Part primitives and boolean operations (fuse, cut, common)
- Draft operations including ShapeString for extrudable 3D text
- CAM toolpath generation (profile, pocket, adaptive, drilling) with G-code post-processing
- Mesh import/export and solid conversion
- Spatial queries: interference, clearance, containment, face relationship, alignment
- Measurement operations: list faces, bounding box, volume, surface area, center of mass
- Spreadsheet-driven parametric models
- Multi-instance support: spawn and manage headless FreeCAD instances
- `execute_python` for arbitrary FreeCAD Python with no timeout (safe for long OCCT operations)
- `api_introspection`: live signature/docstring lookup with fuzzy search across FreeCAD modules
- `macro_operations`: list, read, and run macros from the user's FreeCAD macro directory
- `run_inspector`: design-rule checks against the live document
- Crash recovery: last-operation journal survives FreeCAD crashes; `.FCStd` recovery file cleanup
- Debug infrastructure: structured operation logs, Report View reader, health monitor
- Checkpoint/rollback, section views (clip planes), cross-document shape insertion
